COMO EJECUTAR:


1) Cargas el archivo .asm en MARS.
2) Seleccionas la opcion de assemble, su logo es un destornillador cruzado con una llave inglesa.
3) En la pestaña de tools, seleccionas la opcion de Bitmap Display.
4) Esto te abrira una pestaña, en la cual debes asegurarte que el Display Width in Pixels y el Display Height in Pixels tengas la opcion de 512 antes de cargar. Asegurate tambien que Base address for display tenga seleccionada la opcion de 0x10010000 (static data).
5) Una vez hecho esto, podes darle a la opcion de Connect to MIPS.
6) Nuevamente en MARS, le das a la opcion de Run, que esta a la derecha de Assemble.
7) El juego esta listo para ser disfrutado. Que tengas una partida llena de emocion.
